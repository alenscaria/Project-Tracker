# Project-Tracker

- Create projects with client name, status, hourly rate, budget, start date & deadline
- Cycle through statuses (active → paused → done → overdue) with one click
- Sidebar shows hours + earned at a glance

- To-Dos - tasks with High/Med/ Low priority, checkboxes, and auto-count of remaining
- Costs - full cost summary (rate hours, expenses, net), budget progress bar
- Notes - freeform notepad that auto-saves as you type

- Dashboard header - shows total earned, total hours, and project count across all projects.
- All data persists in localStorage - no backend needed.
